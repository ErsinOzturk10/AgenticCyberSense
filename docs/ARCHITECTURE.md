## Adding a New Agent

To add a new agent to the system:

```python
# 1. Create new file: src/agenticcybersense/agents/my_agent.py

from agenticcybersense.agents.base import BaseAgent
from agenticcybersense.agents.registry import register_agent
from agenticcybersense.schemas.messages import AgentRequest, AgentResponse

@register_agent  # This automatically registers the agent
class MyNewAgent(BaseAgent):
    name: str = "my_agent"  # Unique identifier
    description: str = "Description of what this agent does"
    
    async def process(self, request: AgentRequest) -> AgentResponse:
        # Implement your agent logic here
        
        return AgentResponse(
            content="Agent response content",
            agent_name=self.name,
            success=True,
            findings=[],  # Add Finding objects here
        )

# 2. Import in agents/__init__.py
from agenticcybersense.agents.my_agent import MyNewAgent

# 3. Update routing in graph/build_graph.py if needed
```
